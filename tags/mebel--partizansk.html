Один из важных моментов при проектировании

Один из важных моментов при проектировании кухни - это выбор сушки для посуды.
Выдвижные сушки.
По форме и конфигурации могут быть разными и предназначены для тарелок, ложек, стаканов с чашками, вообще быть многофункциональными.
Также можно выбрать систему "карусель" , в которой вместо корзин устанавливают вращающиеся полочки или встроить колонну с большим количеством корзин.
В угловой стол установить "волшебный уголок" с выезжающими корзинами.
Встроенная сушка монтируется при сборке кухонного гарнитура.
Стоит такая конструкция несколько дороже...
